- Added ComfyUI LiteGraph workflow import with subgraph support (#12)

- Fixed LoRA chain display for video workflows
- Fixed "UI: X" labels for mapped fields
- Fixed missing fields in Generation Bottom Sheet
- Fixed WebSocket auth failures being confused with network errors (#9)
